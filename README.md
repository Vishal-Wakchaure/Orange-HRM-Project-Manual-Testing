## Test Artifacts

### 1. Project Details
Contains basic project information such as:
- Project name
- Module name
- Assigned date
- Build number
- Browser details
- Cross browser testing details

### 2. Test Scenarios
Prepared high level test scenarios for:
- Login page
- Register page

Each scenario contains:
- Test scenario ID
- Test scenario title
- Reference
- Total number of test cases

### 3. Decision Table
A decision table is created for the Login functionality.
It covers different combinations of:
- Valid and invalid username
- Valid and invalid password
- Expected system behavior

### 4. Test Plan
Contains complete test strategy including:
- Scope
- Test approach
- Test environment
- Entry and exit criteria
- Roles and responsibilities
